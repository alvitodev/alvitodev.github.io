## Overview

**alvito.dev** is my personal portfolio website and digital archive, serving as a central hub for showcasing my projects, technical writing, and professional journey. Built with modern web technologies, the site emphasizes performance and user experience.

## Key Features

- **Blog & Articles**: Technical writing on web development, IoT, and software engineering
- **Project Showcase**: Detailed case studies and portfolio pieces
- **Digital Archive**: Organized collection of notes, resources, and learning materials
- **Responsive Design**: Fully responsive layout optimized for all devices
- **Fast Performance**: Leveraging Astro's island architecture for optimal loading speeds

![alvito.dev Interface](/images/projects/alvitodev-interface.png)

## Technical Implementation

### Architecture

The project utilizes **Astro's island architecture** to deliver only the necessary JavaScript to the browser, resulting in exceptionally fast page loads and improved SEO performance.

```astro
// Example: Selective hydration with Astro islands
<WritingDashboard client:load />
<ThemeToggle client:idle />
```

### Technology Stack

- **Framework**: Astro 4.x with static site generation
- **Styling**: Tailwind CSS with custom design tokens
- **Components**: React for interactive UI elements
- **Hosting**: GitHub Pages with custom domain
- **CMS**: Content collection using Astro's content APIs

## Development Journey

Started as a simple portfolio in 2021, the project has evolved significantly:

1. **Phase 1 (2021)**: Basic HTML/CSS portfolio
2. **Phase 2 (2022)**: Migrated to Next.js with MDX support
3. **Phase 3 (2024)**: Transitioned to Astro for better performance
4. **Phase 4 (2025-Present)**: Enhanced with React islands and improved content architecture
