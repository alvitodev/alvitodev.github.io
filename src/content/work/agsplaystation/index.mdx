## Overview

**AGSPlaystation** is a comprehensive rental management system designed for Playstation gaming rental businesses. Developed in C#, the application streamlines rental operations, manages inventory, tracks customers, and handles payment processing. It provides business owners with an efficient platform to manage their rental operations while offering customers a seamless rental experience.

## Business Context

Gaming rental businesses need specialized tools to:

- Track game and console inventory
- Manage customer rentals and returns
- Process payments and late fees
- Generate business reports
- Monitor rental history and trends

AGSPlaystation addresses these needs with an integrated, user-friendly system.

## Business Logic

### Rental Process Flow

```
START
  ↓
[Search Inventory] ← Check available items
  ↓
[Select Items] ← Choose games/consoles
  ↓
[Validate Stock] ← Confirm availability
  ↓
[Enter Customer] ← Existing or new customer
  ↓
[Calculate Fees] ← Base + member discount
  ↓
[Process Payment] ← Validate and process
  ↓
[Generate Receipt] ← Print/Email receipt
  ↓
[Update Inventory] ← Mark items as rented
  ↓
[Create Rental Record] ← Store transaction
  ↓
END
```

### Late Fee Calculation

```csharp
public decimal CalculateLateFee(Rental rental)
{
    if (rental.ReturnDate <= rental.DueDate)
        return 0;

    var daysLate = (rental.ReturnDate - rental.DueDate).Days;
    var lateRatePerDay = 50000; // IDR per day

    return daysLate * lateRatePerDay;
}
```
