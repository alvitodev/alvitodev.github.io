## Overview

**BudgieVB** is a personal cashflow management application developed in Visual Basic, designed to help users track, analyze, and visualize their financial data. The application leverages **ScottPlot** charting library to provide comprehensive financial visualization and data analysis capabilities.

The name "Budgie" is inspired by the budgerigar, a small parrot often kept as a pet—symbolizing the application's role as a personal financial companion.

## Project Goals

- Create an intuitive interface for personal finance management
- Provide powerful visualization tools for financial analysis
- Enable trend analysis and forecasting
- Support multiple account and budget tracking
- Deliver actionable financial insights

## ScottPlot Integration

### Chart Types Implemented

```vb
' Line Chart for Cashflow Trends
Dim linePlot As New ScottPlot.Plot()
linePlot.AddScatter(dates, cashflows)
linePlot.XLabel = "Date"
linePlot.YLabel = "Cashflow (IDR)"

' Bar Chart for Category Spending
Dim barPlot As New ScottPlot.Plot()
barPlot.AddBar(categoryValues, categoryNames)
barPlot.Title = "Spending by Category"

' Pie Chart for Budget Allocation
Dim piePlot As New ScottPlot.Plot()
piePlot.AddPie(budgetValues, budgetLabels)
```

### Key ScottPlot Features Used

- Multiple axis support
- Legend and annotation capabilities
- Real-time chart updates
- Zoom and pan functionality
- Export to image formats
