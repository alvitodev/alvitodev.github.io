Our (hesitantly) first content-based component

This new version of astro-erudite, v1.5.0, introduces our first content-based component: the callout! I was a bit hesitant about adding this component because, frankly, the entire philosophy behind this project was to be as minimalistic as possible‚ÄîI wanted to simply provide boilerplate to remove the ‚Äúbusy work‚Äù factor that often takes away from the writing process.

However, just based on some blog posts I‚Äôve seen that use this template, I felt like there would be a universal desire just to have this component around in the case where it‚Äôd be needed. The primary inspiration came when user @rezaarezvan sent in a PR to add their site, rezaarezvan.com, to the examples section in the README. They had years upon years of accumulated notes and resources on their site, most of which were in the form of LaTeX-style academic content that requires ‚Äúblocks‚Äù for things like definitions, theorems, and proofs. I sent in an encouraging reply to the PR and then started building the component:

[@jktrn] your blog posts are literally insane btw @rezaarezvan how have you written multiple textbooks worth of educational resources???

i think i might add latex-style theorem/lemma/corollary/def/proof/eg/ex/remark blocks to astro-erudite so i can accommodate for these academia-style blogs, like e.g. for exercises it‚Äôd just be a component with an expandable section to hide the answer

How does it work?

I‚Äôve added a simple Callout.astro to src/components that now comes shipped with the template. It‚Äôs a very easy-to-read component that has an insanely long configuration scheme for all of the different types of callouts that I‚Äôve added. Fundamentally, it follows the same paradigm as shadcn/ui which uses class-variance-authority to create different ‚Äúvariants‚Äù on top of a base styling scheme:

As such, if you feel like there‚Äôs any variant that you‚Äôre missing, it‚Äôs insanely trivial to add it yourself. It‚Äôs less trivial, however, to figure out what colors to use since I‚Äôve essentially taken up every good Tailwind color for these!
